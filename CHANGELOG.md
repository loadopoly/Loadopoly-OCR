# Changelog

All notable changes to this project will be documented in this file.

## [1.7.0] - 2025-01-08

### Added
- **Engine Upgrade:** Migrated to `gemini-3-flash-preview` for world-class OCR and relational mapping speed.
- **PWA Integration:** Added full Progressive Web App support with offline caching via Service Worker.
- **Atomic Architecture:** Flattened project structure to root-relative imports for improved module resolution.
- **Enhanced Documentation:** Comprehensive README update reflecting latest features and architectural shift.

## [1.6.0] - 2025-01-07

### Added
- **Engine Upgrade:** Migrated to `gemini-2.5-flash` for blazing fast complex text reasoning.
- **Graph Visualizer Panning/Zooming:** Integrated `d3.zoom` into the `GraphVisualizer` component.
- **Structured DB Clusters:** Added a hierarchical "Cluster View" to the database tab.

### Changed
- **Database Default View:** Set "Tabular View" as the default view.
- **Collection Naming:** Integrated Gemini-suggested collection names.

## [1.5.0] - 2025-01-05

### Added
- **Admin Broadcast Console:** Enabled administrators to broadcast data to the network.
- **Community Airdrops:** Marketplace section for "Community Airdrops (Free)".
- **CC0 Licensing Pipeline:** Ingestion pipeline updated to support `CC0`.
